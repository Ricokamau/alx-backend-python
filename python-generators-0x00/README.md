# 📦 Task 0: Getting Started with Python Generators

## 🎯 Objective

Set up the foundational MySQL database and `user_data` table required for testing Python generators in later tasks. Seed the table using sample user records from a CSV file.

---

## 📝 Project Summary

In this task, we:

- Connect to a local MySQL server.
- Create a new database `ALX_prodev` if it doesn't already exist.
- Create a `user_data` table with the following fields:
  - `user_id` (Primary Key, UUID, Indexed)
  - `name` (VARCHAR, NOT NULL)
  - `email` (VARCHAR, NOT NULL)
  - `age` (DECIMAL, NOT NULL)
- Populate the table using sample user records from a CSV file: `user_data.csv`.
